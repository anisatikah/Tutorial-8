Q.3(a)
DISCRETE RV · VERIFY
CLO 1
Given the probability distribution function:
$f(x) = \begin{cases} \dfrac{2x^2}{20}, & x = 2 \\ \dfrac{7-2x}{10}, & x = 1, 3 \end{cases}$
Show that X is a discrete random variable.
Q.3(a)
SOLUTION
CLO 1
Calculate each probability $P(X=1) = \dfrac{7-2(1)}{10} = \dfrac{5}{10} = \dfrac{1}{2}$
$P(X=2) = \dfrac{2(2)^2}{20} = \dfrac{8}{20} = \dfrac{2}{5}$
$P(X=3) = \dfrac{7-2(3)}{10} = \dfrac{1}{10}$
Sum all probabilities $\sum P(X=x) = \dfrac{1}{2} + \dfrac{2}{5} + \dfrac{1}{10} = \dfrac{5+4+1}{10} = 1$
Conclusion Since $\sum P(X=x) = 1$ and all $P(X=x) \geq 0$, X is a discrete random variable. ✓
Q.3(b)(i)
DISCRETE RV
CLO 1
Find the probability that X is less than 3.

Recall: $P(X=1) = \dfrac{1}{2}$, $P(X=2) = \dfrac{2}{5}$, $P(X=3) = \dfrac{1}{10}$
Q.3(b)(i)
SOLUTION
CLO 1
Method 1: Direct addition $P(X<3) = P(X=1) + P(X=2)$ $= \dfrac{1}{2} + \dfrac{2}{5} = \dfrac{5}{10} + \dfrac{4}{10} = \dfrac{9}{10}$
Method 2: Complement $P(X<3) = 1 - P(X=3) = 1 - \dfrac{1}{10} = \dfrac{9}{10}$
ANSWER
$\dfrac{9}{10}$
Q.3(b)(ii)
CDF
CLO 1
Find the cumulative distribution function of X.

$F(x) = P(X \leq x)$ — accumulated probability
Q.3(b)(ii)
SOLUTION
CLO 1
Calculate F(x) at each point $F(1) = P(X \leq 1) = P(X=1) = \dfrac{1}{2}$
$F(2) = P(X \leq 2) = \dfrac{1}{2} + \dfrac{2}{5} = \dfrac{9}{10}$
$F(3) = P(X \leq 3) = \dfrac{1}{2} + \dfrac{2}{5} + \dfrac{1}{10} = 1$
ANSWER — CDF
$F(x) = \begin{cases} 0, & x < 1 \\ \dfrac{1}{2}, & 1 \leq x < 2 \\ \dfrac{9}{10}, & 2 \leq x < 3 \\ 1, & x \geq 3 \end{cases}$
Q.4(a)
CONTINUOUS RV · CDF
CLO 1
T = time spent on homework. CDF of T:
$F(t) = \begin{cases} 0, & t < 0 \\ k(2t^3 - t^4), & 0 \leq t \leq 1.5 \\ 1, & t > 1.5 \end{cases}$
Show that $k = \dfrac{16}{27}$.
Q.4(a)
SOLUTION
CLO 1
Use the property of CDF At the upper limit, $F(1.5) = 1$
Substitute t = 1.5 $k(2(1.5)^3 - (1.5)^4) = 1$ $k\left[2 \cdot \dfrac{27}{8} - \dfrac{81}{16}\right] = 1$

Q.4(b)
PDF FROM CDF
CLO 1
Find the probability density function $f(t)$ of T.

Key relationship: $f(t) = F'(t)$ — differentiate CDF to get PDF
Q.4(b)
SOLUTION
CLO 1
Apply $f(t) = F'(t)$ for each piece For $t < 0$: $f(t) = \dfrac{d}{dt}(0) = 0$
For $0 \leq t \leq 1.5$ $f(t) = \dfrac{d}{dt}\left[\dfrac{16}{27}(2t^3 - t^4)\right] = \dfrac{16}{27}(6t^2 - 4t^3)$ $= \dfrac{32}{27}(3t^2 - 2t^3)$
For $t > 1.5$ $f(t) = \dfrac{d}{dt}(1) = 0$
ANSWER — PDF
$f(t) = \begin{cases} \dfrac{32}{27}(3t^2 - 2t^3), & 0 \leq t \leq 1.5 \\ 0, & \text{otherwise} \end{cases}$
Q.5
PDF FROM CDF
CLO 1
The CDF of a continuous random variable Y is given by:
$F(y) = \begin{cases} 0, & y < 0 \\ 2y - y^2, & 0 \leq y \leq 1 \\ 1, & y > 1 \end{cases}$
Find the probability density function of Y.
Q.5
SOLUTION
CLO 1
Differentiate each piece: $f(y) = F'(y)$ For $y < 0$: $f(y) = \dfrac{d}{dy}(0) = 0$
For $0 \leq y \leq 1$ $f(y) = \dfrac{d}{dy}(2y - y^2) = 2 - 2y$
For $y > 1$ $f(y) = \dfrac{d}{dy}(1) = 0$
ANSWER — PDF
$f(y) = \begin{cases} 2 - 2y, & 0 \leq y \leq 1 \\ 0, & \text{otherwise} \end{cases}$

Q.8(a)
PIECEWISE PDF
CLO 2
The PDF of a continuous random variable is:
$f(y) = \begin{cases} py, & 0 \leq y < 1 \\ \dfrac{p}{4}, & 1 \leq y < 3 \\ 0, & \text{otherwise} \end{cases}$
Find the value of $p$.
Q.8(a)
SOLUTION
CLO 2
Set up integral over both pieces $\displaystyle\int_0^1 py\, dy + \int_1^3 \dfrac{p}{4}\, dy = 1$
Evaluate each integral $p\left[\dfrac{y^2}{2}\right]_0^1 + p\left[\dfrac{y}{4}\right]_1^3 = 1$ $p\left(\dfrac{1}{2}\right) + \dfrac{p}{4}(3 - 1) = 1$
Simplify $\dfrac{p}{2} + \dfrac{p}{2} = 1 \quad \Rightarrow \quad p = 1$
ANSWER
$p = 1$
